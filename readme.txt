==  Installation  ==
 



1. Upload "vervenia.zip" to your "/wp-content/plugins/" directory.

2. Extract the contents of the "vervenia.zip" file. 

3. Click the Vervenia icon in your WordPress dashboard.

4. Accept the <a href="http://vervenia.com/terms">Vervenia Terms of Service.</a> 
5. The Vervenia Workspace will load automatically.
